In the supplementary materials, the following files are contained.

appendix.pdf: Appendix to the article with proofs and further details for the algorithm. (appendix.pdf, PDF file)
flsa_1.04.tar.gz: R-package ``flsa'' containing code implementing the algorithm described in the article. (flsa_1.04.tar.gz, GNU zipped tar file)
DatasetGeneration.R: R-code for generating the simulated datasets used in the article. (DatasetGeneration.R, R file)
FLLettersExample.R: Sample code for generating the Figure \ref{fig:FLImage}  in the article. (FLLettersExample.R, R file)
